# Site-de-vendas
Esse readme foi feito para orientar a como criar um repositório local e repositório remoto(Github).

## Crie ou selecione a sua aplicação
-1 Inicie o git com o comando:
```bash
  git init
```
 Agora o repositório está criado localmente no seu computador, vc pode modificar o seu código para realizar o primeiro commit após a mudança.
